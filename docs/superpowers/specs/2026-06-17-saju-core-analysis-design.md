# saju-core 분석 엔진 설계

- 날짜: 2026-06-17
- 패키지: `@4-pillars/saju-core`
- 라이브러리: `@fullstackfamily/manseryeok@^1.0.8`

## 목표

생년월일시(양/음력) + 성별을 입력하면 다음을 계산하는 테스트 가능한 명리 엔진:
사주팔자(원국), 오행 분포, 십성 분포, 신강/신약, 용신/희신, 대운.

## 라이브러리 경계

`manseryeok`가 제공: **원국(사주팔자)** = `calculateSaju(y,m,d,h,min,opts)` → 각 주가 `"갑자"`(천간+지지) 한글 2글자 + 한자, 진태양시 보정. 그 외 `solarToLunar/lunarToSolar`, 60갑자(`SIXTY_PILLARS`, `getPillarByHangul`), 절기 함수.

**제약**: 정밀 절기(`getSolarTermsByYear`)는 2020~2030만 지원. 게다가 검증 결과 이 함수는 **연도와 무관하게 동일한 고정(명목) 시각**을 반환함이 확인되어 정확한 연도별 절기로 쓸 수 없다. 따라서 대운수(전 범위 1900~2050 필요)를 위해 **절기 일시를 태양황경 기반(Meeus 근사)으로 자체 계산**한다. 검증은 라이브러리 비교 대신 솔버 자기일관성(구한 순간의 황경=목표값)과 알려진 실제 KST 값 근사로 한다.

원국 외 분석(오행/십성/신강/용신/대운)은 모두 자체 구현.

## 방법론 (확정)

- **지지 취급**: 본기(正氣)만 — 각 지지를 대표 오행/대표 천간 1개로 환원.
- **신강/신약·용신**: 억부법.

## 알고리즘 요약

### 오행 분포
8자리(천간4 + 지지 본기4) 각 오행 집계 → count + percent.

### 십성 (일간 D 기준, 상대 천간 E)
- 동오행: 동음양=비견 / 이음양=겁재
- D生E: 동=식신 / 이=상관
- D克E: 동=편재 / 이=정재
- E克D: 동=편관 / 이=정관
- E生D: 동=편인 / 이=정인
지지는 본기 천간으로 분류.

### 신강/신약 (억부 점수)
일간 제외 7자리에 가중치(월지3·일지2·년지1.5·시지1.5·천간 각1) 부여. 십성을 5세력으로 묶어
우호(비겁+인성) vs 비우호(식상+재성+관성) 합산. strength = 우호/(우호+비우호).
≥0.55 신강 / ≤0.45 신약 / 그 사이 중화. 가중치·임계값은 튜닝 상수.

### 용신/희신 (억부)
5세력 강도 비교 후:
- 신약: 비우호 최강이 재성 → 용신=비겁(=D), 그 외 → 용신=인성(D를 생하는 오행)
- 신강: 우호 최강이 인성 → 용신=재성(D가 극하는 오행), 그 외(비겁) → 용신=식상(D가 생하는 오행)
- 희신 = 용신을 생하는 오행.

### 대운
- 방향: 양남음녀 順, 음남양녀 逆 (년간 음양 × 성별).
- 대운수: 순행=출생→다음 節氣, 역행=출생→이전 節氣 까지 일수 ÷ 3 (반올림). 節氣 일시는 자체 황경 계산.
- 간지: 월주 id에서 ±1(순/역) 씩 60갑자 전개, 10운, 각 10년.

## 테스트 (TDD, vitest)

- 십성/오행: 알려진 입력(예 1990-05-15 14:30)으로 표 검증.
- 절기 황경 계산: 솔버 자기일관성(< 1e-4°) + 입춘 2024 ≈ 실제 KST 값.
- 대운: 방향/대운수/간지 시퀀스 스냅샷.
- 검증: `pnpm --filter @4-pillars/saju-core test`, `demo`로 전체 출력.

## 범위 밖
조후/통관/병약 용신, 지장간 전체 가중, 신살/공망, 합충형파해.

## 향후 발전 방향

현재 버전은 "테스트 가능한 단순화 억부 모델"이다. 정밀도·해석 깊이를 높일 여지를 영역별로 정리한다.

### 1. 원국 정밀도

- **야자시(夜子時)/조자시 처리**: 23:00~01:00의 일주·시주 경계 처리 옵션. 현재는 라이브러리 기본값에 의존.
- **경도/서머타임 보정 확장**: 출생지 경도 입력 UI 연동, 1948~1988년 한국 서머타임·표준시 변경 이력 반영. 일본 서비스용 일본 표준시(135°) 분기.
- **양력 범위**: 라이브러리는 1900~2050. 그 밖 연도 요청 시 명확한 에러/안내.

### 2. 지지 정밀화 (현재 본기만)

- **지장간 전체 가중**: 여기(餘氣)·중기(中氣)·정기(正氣)를 비율 가중해 오행/십성/신강 집계. (방법론 질문에서 보류한 옵션 — 데이터 테이블에 지장간·일수 추가하면 확장 가능)
- **월령 사령(司令)**: 절입일로부터 경과일에 따라 어느 지장간이 당령하는지 반영 → 신강/신약 정밀도 향상.
- **왕상휴수사(旺相休囚死)**: 월령 기준 오행별 계절 강약 계수.

### 3. 신강/신약 고도화

- **득령·득지·득세 분리 판정**: 현재 단일 가중점수 → 3요소 개별 표기.
- **가중치·임계값 캘리브레이션**: `POSITION_WEIGHTS`/`STRENGTH_THRESHOLDS`를 실제 사례 데이터셋으로 검증·조정. 중화 구간(0.45~0.55) 세분화.
- **합충(合沖)에 따른 세력 가감**: 천간합·지지 삼합/육합/충이 일간 세력에 미치는 영향.

### 4. 용신 다층화 (현재 억부만)

- **조후용신**: 월지(계절)·일간 조합표로 한난조습 보정. (브레인스토밍에서 보류)
- **통관·병약용신**: 대립 오행 사이 통관신, 기신을 제거하는 약신.
- **격국(格局) 판정**: 월지 중심 정격/외격 분류 후 격에 맞는 용신.
- **다중 용신 후보 + 신뢰도**: 단일 결정 대신 후보별 근거·점수 제시.

### 5. 대운/운세 확장

- **대운수 정밀 표기**: 현재 반올림 정수 → 년·월·일 단위(3일=1년, 1일=4개월) 환산.
- **세운(歲運)**: 특정 연도 간지와 원국·대운의 상호작용.
- **월운/일운**: 절기 기반 월주, 일진 계산(라이브러리 `getGapja` 활용).
- **대운-원국 상호작용**: 각 대운의 십성·용신 적부(吉凶) 해석 데이터.

### 6. 관계·구조 분석 (신규 모듈)

- **합충형파해(合沖刑破害)**: 천간합·지지 합/충/형/파/해 탐지.
- **신살(神殺)**: 도화·역마·천을귀인·공망 등.
- **십이운성(十二運星)**: 일간 기준 각 지지의 장생~절.

### 7. 엔지니어링

- **`@4-pillars/types`로 공통 타입 승격**: `Element`/`Stem`/`Branch`/`SajuChart` 등을 타입 패키지로 올려 report-engine·content·ui와 공유.
- **계산 결과 스냅샷 테스트 세트 확대**: 다양한 생년월일(경계·윤달·자시) 케이스 픽스처화.
- **절기 계산 고정밀화**: 필요 시 Meeus 고정밀(주기항) 또는 VSOP87로 교체(현재 저정밀 ~분 단위로 충분).
- **성능**: 대운 절기 계산 메모이즈(현재 1회 36 솔브, 충분히 빠름).
- **report-engine 연동**: `SajuAnalysis` → 룰 기반 콘텐츠 키 매핑(일간+십성+신강 조합)으로 리포트 조립.
