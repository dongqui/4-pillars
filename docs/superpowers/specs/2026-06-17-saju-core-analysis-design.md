# saju-core 분석 엔진 설계

- 날짜: 2026-06-17
- 패키지: `@4-pillars/saju-core`
- 라이브러리: `@fullstackfamily/manseryeok@^1.0.8`

## 목표

생년월일시(양/음력) + 성별을 입력하면 다음을 계산하는 테스트 가능한 명리 엔진:
사주팔자(원국), 오행 분포, 십성 분포, 신강/신약, 용신/희신, 대운.

## 라이브러리 경계

`manseryeok`가 제공: **원국(사주팔자)** = `calculateSaju(y,m,d,h,min,opts)` → 각 주가 `"갑자"`(천간+지지) 한글 2글자 + 한자, 진태양시 보정. 그 외 `solarToLunar/lunarToSolar`, 60갑자(`SIXTY_PILLARS`, `getPillarByHangul`), 절기 함수.

**제약**: 정밀 절기(`getSolarTermsByYear`)는 2020~2030만 지원. 게다가 검증 결과 이 함수는 **연도와 무관하게 동일한 고정(명목) 시각**을 반환함이 확인되어 정확한 연도별 절기로 쓸 수 없다. 따라서 대운수(전 범위 1900~2050 필요)를 위해 **절기 일시를 태양황경 기반(Meeus 근사)으로 자체 계산**한다. 검증은 라이브러리 비교 대신 솔버 자기일관성(구한 순간의 황경=목표값)과 알려진 실제 KST 값 근사로 한다.

원국 외 분석(오행/십성/신강/용신/대운)은 모두 자체 구현.

## 방법론 (확정)

- **지지 취급**: 본기(正氣)만 — 각 지지를 대표 오행/대표 천간 1개로 환원.
- **신강/신약·용신**: 억부법.

## 알고리즘 요약

### 오행 분포
8자리(천간4 + 지지 본기4) 각 오행 집계 → count + percent.

### 십성 (일간 D 기준, 상대 천간 E)
- 동오행: 동음양=비견 / 이음양=겁재
- D生E: 동=식신 / 이=상관
- D克E: 동=편재 / 이=정재
- E克D: 동=편관 / 이=정관
- E生D: 동=편인 / 이=정인
지지는 본기 천간으로 분류.

### 신강/신약 (억부 점수)
일간 제외 7자리에 가중치(월지3·일지2·년지1.5·시지1.5·천간 각1) 부여. 십성을 5세력으로 묶어
우호(비겁+인성) vs 비우호(식상+재성+관성) 합산. strength = 우호/(우호+비우호).
≥0.55 신강 / ≤0.45 신약 / 그 사이 중화. 가중치·임계값은 튜닝 상수.

### 용신/희신 (억부)
5세력 강도 비교 후:
- 신약: 비우호 최강이 재성 → 용신=비겁(=D), 그 외 → 용신=인성(D를 생하는 오행)
- 신강: 우호 최강이 인성 → 용신=재성(D가 극하는 오행), 그 외(비겁) → 용신=식상(D가 생하는 오행)
- 희신 = 용신을 생하는 오행.

### 대운
- 방향: 양남음녀 順, 음남양녀 逆 (년간 음양 × 성별).
- 대운수: 순행=출생→다음 節氣, 역행=출생→이전 節氣 까지 일수 ÷ 3 (반올림). 節氣 일시는 자체 황경 계산.
- 간지: 월주 id에서 ±1(순/역) 씩 60갑자 전개, 10운, 각 10년.

## 테스트 (TDD, vitest)

- 십성/오행: 알려진 입력(예 1990-05-15 14:30)으로 표 검증.
- 절기 황경 계산: 솔버 자기일관성(< 1e-4°) + 입춘 2024 ≈ 실제 KST 값.
- 대운: 방향/대운수/간지 시퀀스 스냅샷.
- 검증: `pnpm --filter @4-pillars/saju-core test`, `demo`로 전체 출력.
